POSCOOP

Poscoop merupakan aplikasi semi-erp yang digunakan untuk mengolah data di Swalayan atau Toko. Berbasis web menggunakan PHP Native dan database MySQL. Disini saya berperan sebagai IT Technical Support dengan jobdesk melakukan instalasi server, client dan jaringan untuk penggunaan aplikasi, melakukan training penggunaan aplikasi dengan client, membuat manual book aplikasi dan mengatasi trouble atau penambahan fitur pada aplikasi poscoop.
